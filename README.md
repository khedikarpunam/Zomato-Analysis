# Zomato Restaurant Data Analysis

## Abstract

This project performs Exploratory Data Analysis (EDA) on the Zomato restaurant dataset using Python. The analysis focuses on understanding restaurant ratings, online ordering, table booking, cuisines, pricing, and customer preferences through data cleaning and visualization.

---

## Keywords

Python, Data Analysis, Exploratory Data Analysis (EDA), Pandas, NumPy, Matplotlib, Seaborn, Zomato Dataset

---

# 1. Introduction

The food industry generates large amounts of customer and restaurant data. This project analyzes the Zomato dataset to discover useful insights about restaurant ratings, pricing, cuisines, and online ordering services.

---

# 2. Problem Statement

Restaurant datasets contain thousands of records that are difficult to analyze manually. The objective of this project is to clean the dataset, perform exploratory data analysis, and visualize important business insights.

# Zomato Dataset Analysis

## About Dataset
This dataset contains information about 
56,252 restaurants from Zomato platform 
with 13 columns including name, location, 
cuisines, rating, and cost.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Banashankari has the most restaurants
- North Indian and Chinese cuisines are most popular
- Most restaurants have ratings between 3.8 to 4.1
- Expensive restaurants tend to have higher ratings
- Most restaurants accept online orders

## Recommendations
1. Focus on Banashankari for partnerships
2. Feature North Indian and Chinese restaurants
3. Add Top Rated badge for 4.0+ rated restaurants
4. Create Premium Dining section
5. Improve online delivery system
